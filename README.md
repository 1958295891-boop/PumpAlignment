# PumpAlignment

机泵对中安卓软件

## 项目简介

这是一个用于机泵对中的 Android 应用程序，提供精确的对中测量和指导功能。

## 功能特性

- 实时对中数据采集
- 对中算法分析
- 可视化展示
- 数据导出

## 技术栈

- Android SDK 33
- Java
- Material Design UI
- AndroidX

## 安装与使用

1. 克隆仓库
   ```bash
   git clone https://github.com/1958295891-boop/PumpAlignment.git
   ```

2. 在 Android Studio 中打开项目

3. 同步 Gradle 文件

4. 编译并运行应用

## 开发环境

- Android Studio
- JDK 1.8+
- Android SDK 33
